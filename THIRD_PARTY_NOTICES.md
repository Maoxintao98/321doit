# 第三方开源鸣谢 / Third-Party Open-Source Acknowledgements

321Doit 是自由开源软件。我们由衷感谢以下开源项目、维护者与社区。

321Doit is free and open-source software. We gratefully acknowledge the
following open-source projects, maintainers, and communities.

## 随应用使用或分发 / Used or distributed with the app

### FFmpeg / FFprobe 8.1.2

- 项目 / Project: <https://ffmpeg.org/>
- 用途 / Used for: 高级媒体探测、转码、LUT 烧录及部分专业格式处理。
  Advanced media inspection, transcoding, LUT bake-in, and selected
  professional-format workflows.
- 许可 / License: LGPL v2.1 or later.

正式离线安装包包含 Universal 2 版本。该构建未启用 `--enable-gpl` 或
`--enable-nonfree`；对应源码、校验和、构建配置与 LGPL 文本位于应用的
`Contents/Resources/ThirdParty/FFmpeg/` 中。

The formal offline installer includes a Universal 2 build. It is built without
`--enable-gpl` and `--enable-nonfree`; corresponding source, checksum, build
configuration, and LGPL texts are available in
`Contents/Resources/ThirdParty/FFmpeg/`.

### OpenCode

- 项目 / Project: <https://github.com/anomalyco/opencode>
- 用途 / Used for: Mira AI 模式的本地后台；也可作为 321Doit 本地 MCP
  服务的兼容客户端。
  Local backend for Mira AI mode; it can also act as a compatible client for
  the 321Doit local MCP server.
- 许可 / License: MIT License, Copyright (c) 2025 opencode.

当构建嵌入 OpenCode 时，精确版本与来源会写入应用的
`Contents/Resources/ThirdParty/OpenCode/BUILD-INFO.txt`。Mira 仅在 Apple
Silicon Mac 上使用这个后台；未嵌入时，它会使用兼容的本机安装。

When a build embeds OpenCode, its exact version and source are recorded in
`Contents/Resources/ThirdParty/OpenCode/BUILD-INFO.txt`. Mira uses this backend
only on Apple Silicon Macs; when it is not embedded, it uses a compatible local
installation.

## 开发与验证鸣谢 / Development and verification

### ascmhl 1.2 / ASC MHL reference implementation

- 项目 / Project: <https://github.com/ascmitc/mhl>
- 用途 / Used for: 测试期交叉验证 321Doit 自行生成的 ASC MHL v2 文件。
  Cross-validating ASC MHL v2 files generated by 321Doit during testing.
- 许可 / License: MIT License, Copyright (c) 2020 ASC.

`ascmhl` 不是 321Doit 的运行时依赖，也不随安装包分发；开发环境可通过
`./Tools/setup_ascmhl.sh` 按固定版本安装。

`ascmhl` is not a runtime dependency of 321Doit and is not distributed with the
installer. The development environment can install the pinned reference version
with `./Tools/setup_ascmhl.sh`.
